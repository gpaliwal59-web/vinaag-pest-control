# Images Folder

Apni photos yahan dalein:

## Folder Structure:

```
images/
├── hero-bg.jpg           (Home page background)
├── services/
│   ├── termite.jpg
│   ├── rodent.jpg
│   ├── cockroach.jpg
│   ├── bedbug.jpg
│   ├── mosquito.jpg
│   └── commercial.jpg
├── about/
│   ├── team.jpg
│   └── office.jpg
└── gallery/
    ├── work1.jpg
    ├── work2.jpg
    └── work3.jpg
```

## Instructions:

1. Apne photos ko appropriate folder mein copy karein
2. File names simple rakhen (no spaces)
3. Recommended formats: .jpg, .png
4. Image size: Under 500KB (compress karein agar zyada hai)

For detailed guide, check: `HOW-TO-ADD-IMAGES.md`
